**Conclusions**
**Task**

У результаті роботи програми було складено розклад занять із використанням жадібного алгоритму для задачі покриття множини.

Алгоритм дозволив мінімізувати кількість викладачів, необхідних для покриття всіх предметів, враховуючи при цьому приорітет тих, хто може викладати більше дисциплін, а в разі однакової кількості - молодших за віком.

Отримали, що для покриття п'яти предметів достатньо трьох викладачів.
Усі дисципліни призначено відповідно до компетенцій викладачів, а розклад охоплює всю множину предметів. Це підтверджує ефективність жадібного підходу для подібних оптимізаційних задач.
